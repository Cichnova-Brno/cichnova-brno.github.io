# vyuzití

Tento bash script, je na dodání znaků non breaking space do podkladů do json textu. Non breaking space se využívá na označení předložek po kterých se nesmí rozdělit řádek.

# používání

Tento script používá standardní programové vybavení jakékoliv linuxové distribuce. Na windows se může využít WLS2. Prostě otevřete terminál v adresáři se skriptem a spusťte ho pomocí ./nbsp.sh
