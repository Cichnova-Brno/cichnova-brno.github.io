# speakers.html

Je to statická HTML stránka, která byla vytvořena skriptem speakers_gen.mjs. Tento soubor je uložen v adresáři /generator.
Spouští se příkazem:
    node speaker_gen.mjs

Samotná stránka obsahuje pamětníky a jejich texty, které jsou ve sourorech JSON. Více popsáno v generator.mjs.md.
Pod některými nadpisy jsou videa z Youtube přidaná pomocí html tagů iframe.
