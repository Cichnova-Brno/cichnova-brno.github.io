# statické stránky
V kořenovém adresáři jsou HTML soubory, jako jsou books.html, disconnect.html, documentary.html, exhibition.html, index.html a more.html.K těmto stránkam ještě patří původní_vojenský_prostor.html z adresáře vesnice. Tyto soubory nejsou generované skriptem generate.mjs ani speakers_gen.mjs.

Jejich struktura je pevně daná a lze je upravovat pouze v kódu.

# books.html
Tato stránka obsahuje krátké texty ke knihám DONUCENI ODEJÍT a Ptáčata bez hnízda.

# documentary.html
Obsahuje krátký text a iframe na Youtube video.

# exhibition.html
Je na ní shrnutí příběhu z roku 1940 - 1945. Dále tam jsou data a místa další výstavy. Na konci je link na stránky moravského zemského muzea.

# mode.html
Tato stránka obsahuje text s podrobnostmi o projektu a kontakty na p. Danu Vedru, Marii Horákovou.

# původní_vojenský_prostor
Je uložen v adresáři /vesnice. Obsahuje texty k vystěhování vesnic. Stejně jako ostatní stránky z tohoto adresáře má galerii. Data a obrázky pro galerii jsou uloženy v podkladech.
