# využití

Všechny znaky.jpg byly v jiném rozlišení. Tenhle bash script byl napsán, aby všechny byly stejně velké. Tento script byl napsán na jedno použití, ale kdyby někdy měla nastat situace kdy bude znovu potřeba je zamýšlen jenom pro interní pouzití lidma co vědí co dělají!

# používání

Tento script využívá základní vybavení jakékoliv Linux distribuce, plus program imagemagic. Na windows samozřejmě možnost WLS2. Pro použití, otevřete terminál v adresaři se scriptem a spustíte ho pomocí ./rszznak.sh
